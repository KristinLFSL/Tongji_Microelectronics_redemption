# Tongji_Microelectronics_redemption
写在最前，拖了这么久，终于闲下来把这些东西整理出来了，不知道有多少同济微电子学生可以看到呢。  
我并不想给同济微电子浇冷水，但非常不幸的是，如今同济微电子的问题不仅仅出在课程安排上，还有诸如青教不足、方向不清等种种问题，各位权且将我所说的当成一个生存手册吧。  
**课程学习**
最基础的东西，却是同济微电子最缺失的部分。在我看来，济微的课程有两个非常致命的问题：课程设置不合理以及课程质量教学水平有待提高。
第一方面，对于国内现有的集成电路专业方向设置较好的高校，各位也可以自己去调查，以我知道的为例，北京大学集成电路专业是有两个分流方向的，分别是微电子（物理方向）专业和集成系统设计（模拟数字），前者很显然，主要课程围绕半导体物理展开，半物、半器、工艺，以及前置课程固体物理、量子力学等是主要课程，有关电路设计的只会浓缩为一门4学分的课程，未来这部分学生相当一部分也会去到器件方向，非常对口；后者课程则主要围绕电路设计的相关课程展开，模电数电模集数集eda等课程，对于理论物理部分仅设置一门4学分的半导体基础课程，未来这部分学生可以去到模拟、数字方向，也可以去到eda方向。而济微的学生，却要苦逼地将这两部分杂糅在一起学，如果有济微的老师看到我的这段文字，或许您也会认同我，这两个大方向看似有关，但其实两者都仅需要了解对方的一点基础知识即可，器件只需要知道期望的器件特性和电路功能，而设计只需要知道器件的表征特性或者说对于器件工作点的应用，专业内的分流，迫在眉睫，或许微电子学院真正成立的那一天真的会实现吧。
在学习课程如此繁杂的情况下，济微的课程内容与质量也是参差不齐。据我了解，不管是清北还是国外高校如台湾大学等，现在已经不会再讲述以BJT为基础的模拟电子技术了，大部分集成电路强校所有的教学都是基于MOS展开的（突击检查，MOSFET的全称是什么，这是真的有可能面试会考的），通过模电和模集两门课程基本会把拉扎维那本书讲完，真希望未来济微可以进行改革，ql老师就靠你了；数字方面，其他高校都有成体系的开发设计类课程，也有Verilog代码的课程学习，前者我认为，校内的gy老师、rhq老师、zzf老师等开设的几门如微控制器、嵌入式课程都非常不错，干货很多非常硬核；但是后者，就需要各位的自学了，学校目前除高程之外不会再外设更多的专业编程语言的学习课程，这类编程语言是前者的基础，只有掌握这类语言才能熟练的应用。说来惭愧，在济微的这几门相关课程，我都是被同学带飞的，导致我直到毕业也不太会Verilog哈哈，如果需要驱动力的话，可以大二开始报一些开发类的比赛，具体可以查询学院的竞赛名录，假如能拿奖，还可以给自己加分，就算没拿奖，也自驱学会了一些开发技能，无论如何对个人发展都非常有利；器件方面，其实学校的本科教育有些亏欠，但相关方面的师资在近两年快速充盈，假如同学有志于此方向，在课程学习上，请原谅我的尖锐，校内的几门相关课程可能只有温老师的微电子工艺水平在线，对于关键的半物和半器，我建议大家直接打开玉龙大片或者西电柴老师的课程，收获颇丰，对于零基础的同学也比较友好，同时在集创赛等一些竞赛当中，也会有器件仿真类型的赛题，也可以借此学习一些TCAD软件工具，这个技能非常重要，对于本科生来说掌握TCAD工具可以让未来导师或者上司眼前一亮。
此外，我必须和各位说明，济微现在设置的很大一部分课程，对于专业主流的就业和升学方向来说毫无作用，纯粹的浪费时间，以我在济微几年的见闻，大概率得归因于教研工作不到位以及专业内几位老师的竞争。我实在不清楚济微当中设置那么多天线微波方面课程的原因，或许是出自某位Fellow的大手吧，荼毒了一代又一代济微人，浪费了大量的时间。我并不是说这些课程不该开设，但它不应该作为必修，让需要的人学习，本来济微的课程杂糅就导致了课业压力极大，课改迫在眉睫。
**学校资源及现有方向**
虽然济微的师资体量不大，但可贵的是方向比较齐全，模拟方向有ql、mm老师，数字方向虽然没有很权威的老师，但也有选择，eda有hq老师，器件方向今年来引进的wpy老师人也非常nice，对于这些老师，我不做过多的评价，劳累程度、方向热度和未来薪资这些东西，本身就是需要trade off的
**就业导向**
说实在的，我已经见到太多济微就业不尽如人意转行的例子了，济微目前的专业竞争力在28所中确实较低，可能仅靠一个985的头衔硬撑。我这里的就业，仅针对于专业内的方向建议。对于想要直接就业的同学，最可靠的路径就是自学开发，可以通过大量的竞赛来积攒经验学习技能，没办法，如果去模拟，现在本科可能都找不到layout岗位，而且本身模拟门槛较高。除了竞赛外，本科期间的实习也可以通过海投获得，重点关注张江那边的企业，企业层次是次要，重点是经历，实习就是你有了第一段才会有第二段，只要资质齐全，小企业也可以考虑。
**升学导向**
升学无非考研和保研两条路径，令人欣慰的是，济微的保研率近几年一直全校靠前，主要得益于国家对于这类专业的扶持。如果你想保研，最平常的办法就是绩点卷到前十名，这个名次基本可以保证在最后的综合排名当中也可以获得保研名额，同时rk前十也有较大的概率去到清北复交等学校；其次还可以通过竞赛加分，具体参考前文，此外还可以通过科研加分，论文相较于比赛，周期更长，但却可以让你收获科研技能，对于济微的学生，比较可行的可能就是wpy老师的组内科研了，当然对于不怕学术污点的各位邪修，也可以选择去找系内的那位功利地发一篇论文，现在想来，其实也没有什么指摘别人的意味，这些都是个人的取舍，一篇无足轻重的论文，换来一个保研名额，总还是值得的。至于考研，我也不敢瞎说，但我会在这里尽量为大家收集我的同学的考研资料，希望对大家有帮助。
建这么一个库的目的，主要就是让济微的学生不要那么的痛苦，可以有更多的时间选择自己想走的路，这些都是我曾经的课程作业或者是学习资料，或是粗糙或是详细，希望对各位有用。
说实话，我曾经也像各位一样，为了收集信息焦头烂额，一份往年题可能就能高兴好几天。说真的本科的水很深，对于一些不是那么敏感的同学，机会转瞬即逝，甚至有些学生还会利用职务之便为自己谋利，由于公开，我并不能在这里讨论这些，我能做的就是把我有的东西放在这里，提供给所有人，也希望后人可以一直传下去吧。
